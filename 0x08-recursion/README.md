# 0x08. Recursion

## Description
This project focuses on learning **recursion in C**. 
It includes functions that perform tasks without using loops, relying solely on recursive calls.

## Files
- `_puts_recursion.c` - function that prints a string followed by a newline
- `_putchar.c` - provided function to output characters
- `main.h` - header file containing function prototypes
- `0-main.c` - test file provided by ALX

## Compilation
To compile your code, use:
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 _putchar.c 0-main.c 0-puts_recursion.c -o 0-puts_recursion
