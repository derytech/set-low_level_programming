# 0x03-debugging - Positive or Negative Test

## Description

This directory contains C programs related to debugging exercises. The main task here is to **test the function `positive_or_negative()`** with a specific case.

- `0-main.c` - Tests the function with the value `0`.
- `main.h` - Header file declaring the `positive_or_negative()` function.

## Usage

1. Compile the test program with the function implementation (provided by the platform):

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 positive_or_negative.c 0-main.c -o 0-main
